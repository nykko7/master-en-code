Día 4:

Box Model

  • Propiedades de CSS:
    • Border: Agrega borde al contenido de una etiqueta.
    • Margin: Espacio desde el borde hacia afuera.
    • Padding: Espacio desde el borde hacia adentro.

  • Display de elementos:
    • block: Permite modificar alto y ancho, pero no permite un elemento al lado.
    • inline: No permite modificar alto y ancho, pero si permite un elemento al lado. También se puede agregar margin(left y right) y padding.
    • inline-block: Combinacion de ambas. Permite modificar ancho y altura y permite elementos al lado.

Día 5:

Flex Box

  • Contexto: Apple lanza navegador web para móviles.
  • Responsive Design: Adaptar la apariencia de la web para distintos dispositivos.
  • Flex Box: Flexible Boxes: contenedores que se adaptan a la pantalla del dispositivo automáticamente.
  • Propiedades:
    • justify-content - alineación eje x: En Acción
      align-items - alineación eje y: En Acción
      • center: alineación al centro del contenedor.
      • flex-start: alineación al comienzo del contenedor.
      • flex-end: alineación al final del contenedor.
      • space-between: los elementos se distribuyen con espacios entre los elementos.
      • space-around: los elementos se distribuyen con espacios entre los elementos y los bordes.
    • flex-direction: [column o row]: da dirección a los elementos del contenedor.
